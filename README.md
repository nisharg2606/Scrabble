# Scrabble

This Repo is for making Scrabble game. Please See this [PPT](https://drive.google.com/open?id=1FNwtgogadEQDWEKhIVq2P8qToCxcWsjp) for understanding algorithm and working model.

![](demo.gif)

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

```
Python3
```

### Installing

first clone this git repository or download zip file.

```
git clone https://github.com/milandungrani/scrabble.git
```


Installing dependancies

```
pip install dawg
```
```
pip install texttable
```

Now, go to folder

```
cd scrabble\MainLogic
```

starting script

```
python main.py
````
